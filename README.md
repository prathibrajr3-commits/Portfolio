# Personal Portfolio Landing Page

This project is a simple static portfolio homepage built with HTML and CSS. It introduces **Prathib Raj** as a frontend developer and includes a fixed header, hero section, call-to-action buttons, and social media icons.

## Preview

The page includes:

- A top navigation bar with `Home`, `About`, `Services`, and `Contact`
- A hero section with name, role, short introduction, and action buttons
- Social media icon links using **Boxicons**
- A dark theme with blue highlight colors and basic reveal/hover animations

## Project Structure

```text
Portfolio/
|-- index.html
|-- style.css
|-- black.jpg
```

## Files

- `index.html` contains the page structure and content
- `style.css` contains the layout, colors, hover effects, and animations
- `black.jpg` is currently used as the hero background image
- `.vscode/launch.json` contains Visual Studio Code Edge debugging settings

## How To Run

Because this is a static website, you can run it very simply:

1. Open the project folder in VS Code.
2. Open `index.html` in your browser.

You can also use the VS Code **Live Server** extension if you want automatic refresh while editing.

## Customization

To personalize the portfolio:

- Replace the placeholder paragraph text in `index.html`
- Update the `href="#"` links with real page sections or URLs
- Change the social media links to your actual profiles
- Edit the title in the `<title>` tag
- Update colors, spacing, and animations in `style.css`
- Replace `black.jpg` with your preferred background image if needed

## Notes

- The social icons are loaded from the Boxicons CDN:
  `https://unpkg.com/boxicons@2.1.4/css/boxicons.min.css`
- The current buttons and navigation links are placeholders and do not navigate yet
- The page is designed as a single landing section, not a full multi-page site
